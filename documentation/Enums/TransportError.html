    Indicates the received data was not in the expected format, e.g. invalid JSON.

    Associated error is either JSONDecoderError, an error from JSONSerialization or a custom error thrown from an object’s read(from: JSONObject). The associated error is not used by Forge in any way or displayed in TransportError’s localizedDescription.
