URLTransportable

public protocol URLTransportable: class

Protocol for sending information to a URL.

  • addQuery(key:value:) Extension method

    Adds a query value pair to the query string of the request URL. This value will be appended to the baseURL, path formed by the identifiers followed by a ? in the form key=value. You may add multiple query values before executing a request and they will be separated by the & operator.

    Values are automatically escaped for you.

    Declaration

    Swift

    public func addQuery(key: String, value: URLValue?)

    Parameters

    key

    The left hand side of the query pair.
    value

    The right hand side of the query pair, must be a String, Int or nil. Will be escaped by HTTPRequest.
  • addQueries(_:) Extension method

    Adds a set of query value pairs to the query string of the request URL. These values will be appended to the baseURL, path formed by the identifiers followed by a ? in the form key=value. You may add multiple query values before executing a request and they will be separated by the & operator.

    Values are automatically escaped for you.

    Declaration

    Swift

    public func addQueries(_ queries: Dictionary<String, URLValue?>)

    Parameters

    queries

    A dictionary containing key-value pairs where the key must be an String and the value a URLValue.
  • addHeader(key:value:) Extension method

    Adds a header field to the request. The form in the request will be key: value. No further processing will be performed on the key or value.
